This was one of my favorite projects.

My team worked with NYC street tree data. We cleaned the data, looked for patterns, and tried to understand which trees may cause more sidewalk problems.

I learned a lot about working with messy data and explaining results in a simple way.

This project helped me learn more about AI tools, websites, and backend problems.
