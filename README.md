## 📌 Projects I Worked On

### 🌳 RootCause

RootCause is a risk data analytics dashboard.

My team worked with NYC street tree data. We cleaned more than 684K records and built a dashboard to show patterns and possible sidewalk risk.

This project helped me learn how to work with messy data, build a data pipeline, and explain results in a simple way.

**Tools:** Python, Pandas, XGBoost, Folium, Streamlit

---

### 🤖 Waypoint

Waypoint is a real-time conversational AI project.

I helped build an app that can support voice and text interaction.

This project helped me learn more about APIs, backend systems, and real-time communication.

**Tools:** Claude API, Google Cloud, FastAPI, WebSockets

---

### 🏙️ Mogul Blocks

Mogul Blocks is a real estate strategy game.

My team built a game that simulates real estate market decisions. I learned how to build under time pressure and work with a team during a hackathon.

**Tools:** React, TypeScript, Python, FastAPI, OpenAI API
